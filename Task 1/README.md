# Semantic HTML Personal Profile Page

## Task Description
This project is part of the Web Development Internship Task 1.
The goal of this task is to build a personal profile page using
semantic HTML elements.

## What I Did
- Created a project folder with proper naming conventions
- Wrote HTML boilerplate manually
- Used semantic elements like header, nav, section, article, and footer
- Added personal details such as bio, education, skills, and contact info
- Included a profile image with alt attributes
- Implemented internal navigation using anchor links

## Tools Used
- VS Code
- HTML5
- Google Chrome

## Output
A simple, accessible personal profile webpage built using semantic HTML.
